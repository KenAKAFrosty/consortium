# Plans

Plans are forward-looking and execution-oriented. Keep them scoped, sequenced, and explicit about risks and exit criteria.

## Conventions

- Prefer one file per meaningful workstream.
- Use dated, kebab-case filenames.
- Update the plan when sequencing or assumptions materially change.
- Link related journal entries and decisions as they accumulate.
