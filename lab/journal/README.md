# Journal

Journal entries are compact lab notes. They should record what was tried, what was learned, and what changed because of it.

## Conventions

- Use `YYYY-MM-DD-slug.md` filenames.
- Write an entry when an experiment, review, or discarded direction teaches the project something useful.
- Prefer facts, findings, and consequences over narrative.
- Link to the plan or decision the note influenced.
